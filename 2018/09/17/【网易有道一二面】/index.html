一面

  • 时间：2018 年 8 月 28 日
  • 形式：电话面

自我介绍

略

怼实习

问我实习干了什么？

公司培训了什么？

有没有钱？

聊实习做过的项目，发现没什么亮点，问我做项目时遇到过哪些问题，怎么解决的？

团队中做的贡献。

怼项目

支付宝扫码支付实现思路？

nginx 反向代理原理？

用户和管理员的权限怎么进行控制的？

项目里你怎么进行用户区分？怎么进行会话跟踪的？

session 和 cookie 有什么区别？这两种怎么做会话跟踪？如果一个高并发网站里你会选择用哪一个？

横向越权和纵向越权怎么做的？你这代码重复操作有点多啊，现在能优化一下嘛？

过滤器和拦截器的区别是什么？有什么关系？

给我业务场景设计：多个用户请求并发，有多少种方式实现下单时订单号递增？

锁如何优化？synchronized 和 原子类又有什么区别？熟悉他们底层的机制吗？

问我项目库存设计，面试官当时是看着我的 github 项目问的，如何避免超卖情况出现？除了加锁有更好的方式吗？面试官应该想要我回答用 redis 预读库存，消息队列做请求队列削峰。

问我 mybatis 熟不熟，如何做 mybatis 多条件模糊查询，并且进行分页，说下具体代码思路。

怼基础

进程和线程的区别？

线程如何进行调度？

涉及到操作系统的内核态和用户态会怎么处理？

mysql 事务理解多少？事务的隔离级别有哪几种？

TCP 三次握手四次挥手。

问了 43 分钟，最后问他自己有什么问题要问他的？

二面

  • 时间：2018 年 8 月 30 号
  • 形式：视频面

闲聊

面试官以为是一面，没要求视频面，我提起我一面已经过了，才开始的视频面。算是小乌龙吧。

问项目，有没有上线？有没有具体访问人数？

我：上线了，但是没有人访问。

面试官：嗯嗯，那我们就不问项目了，聊点其他的。。

我：WTF??

数据库

mysql 的引擎了解多少？

innodb 和 myisam 有什么区别？

innodb 的索引了解吗？B-Tree 的原理是什么？为什么索引能提速？如果不用索引为什么慢很多？

mysql 的事务特点讲一下？

（数据库自己真的是挖坑了，好多不会）

进程和线程的区别？

进程怎么调度？操作系统底层做了哪些操作？

手撕 leetcode

看我可能讲的不是很好？那我们来写道算法题吧？

leetcode 第一道题两数相加

题目不是很难，两分钟写了个 O(n2) 的暴力算法，面试官要我优化？

然后讲了一下一次 hash 的思路。面试官说空间复杂度大了，再给一个有序数组，再优化？

我说二分查找？他说二分查找时间复杂度还是大了 O(nlogn) 了，能不能做个时间复杂度 O(n)，空间复杂度为 O(1) 的解决方案？

面试官后面给我讲了快排的思路，让我用快排的思路实现这个题？

太紧张了，没写出来了。晚上自己写了一下，好像特别简单，哎，心态还是不好。。。

撕了半个小时算法吧，涉及到 4 种解决思路，总体来说不难，自己发挥太差了。

最后面试官问我有什么问题要问他？我问了一下部门现在的具体业务是什么，就结束了面试了。

这么久没打电话过来通知三面，我猜应该是凉凉了。
