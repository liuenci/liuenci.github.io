为什么要封装 JsonUtil？

JsonUtil 的作用是可以将一个对象转化为 Json 字符串格式，也可以将一个 Json 字符串格式转化为对象。在缓存系统中不允许存储对象，所以我们可以将对象转化为字符串进行存储，当我们从缓存系统中读取这个字符串的时候，可以将字符串转化为对象。
