自我介绍。

会不会 mongodb? 不会。

会不会 memcached? 不会。

集群环境怎么搭建的？为什么要做集群？多Tomcat+Nginx 搭建，降低服务器压力。

集群除了降低服务器压力还有什么吗？不会。嗯嗯，还有容灾。

反向代理的原理说下？

负载均衡的原理说下？知道哪几种分流算法？轮询，权重，ip_hash,url_hash,fair五种。

nginx 需要隔离出来单独做一台服务器吗？如果 nginx 挂掉能不能访问 Tomcat ?

Session 共享怎么做的？单点登录和单点注销原理说下？

除了用 cookie 做 session 追踪还会什么方法做 session 追踪？不会啊。用 DNS ，你没用过就算了，下一个。

购物车的设计思路和数据库里怎么设计的？用户 id 做外键来标识购物车。

分布式锁了解多少？setnx 是不是原子操作？那 getset 是不是原子操作？

mysql 用过吗？分组的关键字是什么？group by，分组之后能不能再次选择？能啊，用什么关键字？ having 。

分组+选择条件还能再接 order by 进行排序吗？能啊？确定能？确定啊？真的确定？真的确定？这里几个反问问的我心里发毛，现在我都不知道能不能在排序了。

对什么数据结构最了解？hashmap啊。那 hashset 了解吗？卧槽，不会原理，只会用。那你说下 hashmap 的原理吧。

为什么 Java7 中产生哈希碰撞之后要用头插法插到链表里面？因为头插最快啊。

hashmap 怎么根据 key 来获取值。

hashmap 中用了哪种哈希算法？不会。

那你说下 hashmap 和 currentHashmap 的区别吧？分段锁，提高并发量，锁数量决定并发数。

那 hashtable 有用过吗？说下原理。加了 synchronized 的hashmap。

产生死锁的四个条件是什么？循环等待，不可剥夺，占有且等待，最后一个没想出来，还有一个互斥。

知道 arp 吗？知道啊，IP 地址转化为 mac 地址。

那你说下 ARP 欺骗的原理吧？卧槽，不会，我只知道 ARP 欺骗是属于 web 攻击的一种，和 xss,ddos,sql 注入一起的。
