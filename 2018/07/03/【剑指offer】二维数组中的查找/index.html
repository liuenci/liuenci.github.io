【剑指offer】二维数组中的查找

2018-07-03

在一个二维数组中，每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。

思路

利用二维数组由上到下，由左到右递增的规律。

那么选取左下角或者右上角的元素 array[row][col] 与 target 进行比较。

如果 array[row][col] 小于 target 时，那么 target 一定在 第 row 行第 col 列的右边即 col++;

如果 array[row][col] 大于 target 时，那么 target 一定在 第 row 行第 col 列的上边即 row–;
