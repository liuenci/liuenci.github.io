【剑指offer】斐波拉契数列

2018-07-03

大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项,n<=39.

思路

1.首先要理解什么是斐波拉契数列。一个数等于这个数的前两个数的和，这种形式组成的数列就是斐波拉契数列。用公式表示出来就是 f(n) = f(n-1) + f(n-2).

2.这道题一眼看过去，第一个想到的方法就是用递归来写，如果不要求时间复杂度，递归无疑是最快速粗暴的解法了。

3.但是仔细想一下，如果我要计算第 3 个数是多少。那算式就是这样 f(3) = f(2) + f(1) = f(1) + f(1) + f(1)。这样子就是 f(1) 被重复计算了三次，我们如何优化它呢？

4.优化：想一想我们为什么会计算三次 f(1),原因是因为我们没有存储 f(1) 的值导致的，准确的来说，应该是没有存储要计算的数的前面两个数的值。因为要计算某一个数，如果知道前面两个数的值，就很容易就能算出要计算的数了。
