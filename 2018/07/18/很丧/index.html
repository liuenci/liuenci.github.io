2018上半年复盘

2018-07-18

故事要从去年年底说起了，那时候的我对今年的春招抱有很强烈的欲望，因为当时鹏哥说要内推我去饿了么，还有学姐要内推我去蘑菇街和宇宙条，当时兴奋的不行，我对自己说，我一定在寒假里面要好好准备，然后寒假里面办了宽带，复习了一段时间的数据结构。由于自己的进度真的很慢，然后寒假结束，我才复习到二叉树。。

春招算是错过了。

然后时间兜兜转转，一回到学校，被学校里面的各种事情影响，复习进度被严重拖慢，终归是自己自律性不强，没有严格的时间安排，索性就按照学校这边的进度来，决定把 Java 的框架学完，自己另外把驾照考完，今年上半年我觉得自己做的最有成就感的就是把驾照考完了，为我之后的复习流下了很大一块时间。

一直到学校的课程结束，课程设计结束，学校安排的考试以及实习结束，我开始进入自己正式的复习阶段。

七月份我给自己的目标是要学完这些技术：

  1. java 并发，IO，网络编程，集合
  2. linux 命令
  3. redis
  4. spring boot
  5. jvm
  6. nginx
  7. dubbo

一直学习到七月底，八月开始边投简历边复习 github 的秋招文档，相信自己在今年的秋招上面可以找到一个大公司的 offer 的。

唯一觉得难受的是，这段时间自己的精神状态不是很好，负能量爆棚，经常把一个负能量的话发到微博上面，感觉自己像个祥林嫂一样，以后要注意，不能随意吐露自己的难受和焦虑。

小码农最近压力也有点大，晚上经常到凌晨还没睡着，得想想办法缓解一下焦虑，不能一直这么丧着，以后试试晚上跑跑步，锻炼一下腹肌，不能养成啤酒肚才行。

加油。
