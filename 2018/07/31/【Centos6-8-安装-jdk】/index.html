【Centos6.8 安装 jdk】

2018-07-31
查看现有安装的 jdk 版本

在安装完 Centos 之后，默认是有安装过 jdk 的，所以我们需要先查看现有安装的 jdk 版本。
rpm -qa | grep -i java

卸载已有软件

java-1.6.0-openjdk-1.6.0.0-1.66.1.13.0.e16.i686 替换成 rpm -qa | grep -i java 中显示的 jdk 名称
rpm -e –nodeps java-1.6.0-openjdk-1.6.0.0-1.66.1.13.0.e16.i686

上传 jdk

这里可以使用 FileZilla 上传到自己的服务器上。

安装依赖包(需要联网)

yum install glibc.i686

解压 jdk

tar -zxvf jdk-7u71-linux-i586.tar.gz

创建文件夹

mkdir /usr/local/src/java

将 jdk 移动到 /usr/local/src/java

mv jdk1.7.0_72 /usr/local/src/java

配置系统环境变量

先进入一般模式

vim /etc/profile

按 i 进入插入模式

在末尾加上
export JAVA_HOME=/usr/local/src/java/jdk1.7

export PATH=$JAVA_HOME/bin:$PATH

使配置文件生效，并测试

source /etc/profile

java -version

如果能看到 Java 的版本，那么安装成功。
