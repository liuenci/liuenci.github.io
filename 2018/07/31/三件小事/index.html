2018-07-31

总结

没想到日子过的这么快，一下子就到今年 8 月份了。从去年年底开始，我记得那时候区块链大火，我身边的朋友个个炒币，特别是有朋友炒币买房了刺激到了我，于是自己也抱着暴富的美梦，投资了一些区块链项目。凭借着自己看过的关于区块链的书，试图用技术去对抗市场，结果把我套的牢牢的，现在还没解套，从那个时候就觉得自己一夜暴富的美梦破碎了。

但是我毕业总得找份工作维持生活吧，从那个时候我就确定了秋招的想法。从今年三月份开始，我开始认认真真的学习，我清空了脑子里所有不现实的想法，当时给自己定了一份学习路线的计划，时间周期为四个月，也就是 7 月 31 日是我最后一天学习新技术的时间。

当时给自己列了一份技术清单，具体如下：

  1. Java 面向对象基础编程，Java 高级编程如集合，IO，多线程，网络编程，Java8 新特性。
  2. 传统的 Java Web 知识，如 JSP,Servlet。
  3. 三大框架Spring，Spring mvc，Mybatis，外加 Spring boot。
  4. 数据库 sql 语法，oracle，mysql，外加 sql 优化的一部分东西。
  5. 数据结构与算法，用 Java 刷完链表，队列，栈，树，图，用 Java 刷完《剑指offer》中的算法题。
  6. 23 种设计模式中至少要熟悉常见的 7 种。
  7. JVM 内存模型，类加载机制，JVM 调优参数。
  8. 操作系统，计算机网络。
  9. 版本控制 git，项目构建 maven 的使用。
  10. Linux 命令，项目部署，shell 脚本。
  11. 消息队列，缓存。
  12. 做出自己的独立的一个上线项目。

上面列出来这么多技术，很多人觉得四个月的时间很短，完全学习不了这么多东西。我只能说执行力太差，投入的不够多。虽然我也没学完，但是也大概学完了百分之八十左右。因为自己有一部分基础，所以学习起来速度特别的快。

这四个月以来，我看视频学习都是开的两倍速去看的，所以我当时特别容易忘，特别是白天看完视频，配合写代码，到了晚上就差不多忘光了。因为我没有输出，只有输入没有输出的学习很大程度上是无用的学习，所以我选择了白天看视频看书学习，晚上自己写博客总结。

因为版权利益相关，这几个月总结了上百篇的文章无法分享给大家，其中有一部分被我分享在我的博客上，如果你对我的博客有兴趣，欢迎访问 liuenci.cn 帮我添加访问量，还有一部分安安静静的躺在我的有道云笔记当中，后期我可能会考虑做一个电子书版本送给大家。

这四个月晚上经常性的失眠，脑子里真的是各种技术来来去去的在我脑子里面转。所幸的是我坚持过来了，我相信这段时间的学习会给我秋招带来不错的成果。

规划

如果公众号里有准备秋招的小伙伴应该知道，秋招已经开始很久了，今年秋招开始的特别的早。6 月底的时候已经有少部分互联网公司开始了提前批，当时我找了内推，投了广东某游戏公司，过了初筛，过了测评，过了笔试，挂了面试。然后我试着投了一些小公司，发现问的技术没有一点含金量，容易的就过了面试拿了 offer。那时候自己的技术还没有成型，还处在海量吸收的阶段，我觉得自己能去更好的公司，于是给 hr 回了电话，拒了 offer。

从 8 月份开始，我的计划就不是像一个海绵一样疯狂的学习技术了，因为秋招正式开始了。我要投简历了，按照各大互联网招聘的习惯，一般会持续到 11 月底，到时候拿到 offer 之后可以申请提前到公司实习，当然这是建立在学校允许提前出去实习的情况下。

后期规划的话，我打算根据我的简历来进行复习，边复习边投简历。前期可能还是会投一些小公司找找面试经验，有大公司的内推，自己再考虑自身的技术栈是否和对方的招聘要求是否符合进行投递。

每天早上投递完简历之后，每天针对自己的技术栈中的一个小技能去搜索对应的面试题，在复习面试题的同时，还是需要写博客来进行总结与分析，有时候你看别人的东西当时觉得自己会了，但是实际上自己并没有深刻的理解这个技术，所以我认为自己做一些总结是对自己很有用的一个帮助。

公众号

关于微信公众号，我一直在写《剑指offer》的算法题，写这个专栏的目的当初是为了输出倒逼输入，也就是说我为了能让自己每天都有文章可以写，所以会强迫自己去学习。但是算法对大部分人来说还是有门槛的，我也能想到，如果没有面试算法的需求，是不会去看我里面的分析以及代码的，所以《剑指offer》这个专栏我要停更了。

其实我一直有一个想法，就是把自己的公众号当成一个产品来培养。目前自己的目标是在大四毕业之前，达到 1w 粉。现在和我的目标只有一点点就能达到了，这个一点点是多少呢？也就九千九百九十来个吧。

后期我考虑把我手上的一些不错的资源分享给大家，有一些是需要付费才能得到的，有一些是自己总结的，总之省去了大家寻找的时间。另外的想法是 8 月份自己估计会有很多面试的机会，到时候可以在微信公众号上分享自己面试的一些心得，希望我的读者都能斩获 bat 的 offer。
