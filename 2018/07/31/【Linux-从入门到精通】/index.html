后端开发工程师必备的一手潇洒的 Linux 命令。

磁盘管理
ls 命令
  • 功能：列出目录内容
  • 常用
    • ls -l ,此命令非常常用，简化版命令可以使用 ll。
    • ls -al, 查看所有的文件，包括隐藏文件。
cd 命令
  • 功能：切换目录
  • 常用
    • cd ~，切换用户命令
    • cd /，切换到根目录
    • cd ..，切换到上一级目录
    • cd -，切换到上一次访问的目录
pwd 命令
  • 功能：显示工作目录
mkdir 命令
  • 功能：创建目录
  • 常用：
    • mkdir -p xxx，父目录不存在的情况下先创建父目录。
    • mkdir -r xxx，显示命令执行过程中的详细信息。
文件管理-文件浏览
cat 命令
  • 功能：显示文本文件内容
  • 常用：
    • cat xxx，显示指定文件的所有内容。
more 命令
  • 功能：分页显示文件内容，还支持直接跳转行等功能。
  • 常用：
    • more xxx
  • 操作
    • enter，向下 n 行，默认是 1 行。
    • 空格键，向下滚动一屏
    • b ，返回上一屏
    • q，退出
less 命令
  • 功能：分页显示文件内容，操作更详细
  • 常用：
    • less -m，显示类似于 more 命令的百分比
    • less -N，显示每行的行号
  • 操作：
    • 空格键，前进一页
    • b，后退一页
    • d，前进半页
    • u，后退半页
    • enter，前进一行
    • y，后退一行
    • /xxx，向下搜索（用的多）
    • ？xxx，向上搜索（用的多）
    • v，进入 vim 编辑器
    • 左右方向键，相当于水平滚动条
    • q，退出
tail 命令
  • 功能：用于显示指定文件末尾内容。
  • 常用：
    • tail -n，显示 n 行
    • tail -f，循环读取
文件管理-文件操作
cp 命令
  • 功能：复制文件或目录
  • 常用：
    • cp xx1 xx2，复制 xx1 为 xx2,这样只能复制文件。
    • cp -r a b，复制目录一定要加上 -r，当中执行了递归操作。
mv 命令
  • 功能：移动或更名现有的文件或者目录
  • 常用：
    • mv a1 b1 ,修改 a1 变成 b1
    • mv -f a b/a1，如果目标文件或者目录与现有文件或者目录重复，则直接覆盖现有的文件或者目录。
rm 命令
  • 功能：删除文件或者目录
  • 常用：
    • rm xx，删除文件，会有提示是否删除文件
    • rm -f xx，强制删除，没有提示。
    • rm -a xx，删除目录，会有提示是否删除目录
    • rm -rf xx，强制删除目录，没有提示，也就是传说中的删库跑路之必备命令。
find 命令
  • 功能：查找文件或者目录
  • 常用：
    • find -name xxx，查找所有名字为 xxx 的文件或者目录。
    • find /usr/ -name xxx，查找所有在 usr 目录下的文件或者目录。
文档编辑
vi 或 vim 命令
  • 功能：编辑文件
  • 常用：
    • vim xxx，编辑 xxx 文件
  • 常用：
    • vim xxx，进入“一般模式”
    • i，进入“插入模式”
    • esc，退出到“一般模式”
    • 在“一般模式”下，输入“:wq”，保存并退出
管道 |

linux 提供的管道符号 “|” ，作用是“命令1”的输出内容，将作为“命令2”的输入内容，一般和 grep 命令一起使用
格式：命令1 | 命令2

grep
  • 功能：用于过滤/搜索特定字符，可以使用正则表达式能表示多种命令配合使用，使用上十分灵活。
  • 常用：
    • grep -i mysql install.log，查看 install.log 中含有 “mysql” 字符的内容
系统命令
ps 命令
  • 功能：提供对进程的一次性查看。及执行 ps 命令的那个时刻进程信息。
  • 常用：
    • ps -e，显示所有程序
    • ps -f，显示 UID,PPIP 与 STIME 栏位
    • ps -ef | grep -i mysql，配合 grep 显示所有 mysql 的程序
kill 命令
  • 功能：删除执行中的程序或工作
  • 常用：
    • kill -9 4396，强制终止 pid = 4396 的程序
ifconfig 命令
  • 功能：显示网络设备
ping 命令
  • 功能：测试与目标主机的连通性
  • 常用：
    • ping www.baidu.com，测试能否访问百度
    • ping -c 4 www.baidu.com，访问四次之后终止
tar 命令
  • 功能：文件备份压缩
  • 常用：
    • tar -zxvf a.tar，将 a.tar 解压缩
    • tar -zcvf a b，将 a 压缩到 b 目录下
关机重启
  • 常用：
    • 重启：reboot
    • 立即关机：halt
文件权限
chmod 命令
  • 功能：变更文件或目录的权限
  • 常用：
    • chmod u-rwx xxx，取消 xxx 目录用户对其“读写执行”权限
    • chomd g-rwx xxx, 取消 xxx 目录组对其“读写执行”权限
    • chmod 777 xxx，给 xxx 目录添加所有权限
网络配置
网络命令配置
  • 功能：查看网卡配置
  • 常用：
    • cat /etc/sysconfig/network-scripts/ifcfg-eth0，查看网络配置
