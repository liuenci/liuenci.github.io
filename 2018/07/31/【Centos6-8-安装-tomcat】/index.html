【Centos6.8 安装 tomcat】

2018-07-31
上传 tomcat

这里可以使用 FileZilla 上传到自己的服务器上。

安装依赖包(需要联网)

yum install gcc-c++

解压 tomcat

tar -zxvf apache-tomcat-7.0.57.tar.gz

创建文件夹

mkdir /usr/local/src/tomcat

将 tomcat 移动到 /usr/local/src/tomcat

mv apache-tomcat-7.0.57 /usr/local/src/tomcat

启动 tomcat

cd /usr/local/src/tomcat/apache-tomcat-7.0.57/bin

./startup.sh

查看日志

tail -f ../logs/catalina.out

防火墙打开 8080 端口

/sbin/iptables -I INPUT -p tcp –dport 8080 -j ACCEPT

/etc/rc.d/init.d/iptables save
