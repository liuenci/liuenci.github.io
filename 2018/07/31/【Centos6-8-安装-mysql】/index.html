    查看现有安装的 mysql 版本

    在安装完 Centos 之后，默认是有安装过 mysql 的，所以我们需要先查看现有安装的 mysql 版本。
    rpm -qa | grep -i mysql

    卸载已有软件

    使用以下语句进行卸载

    yum remove -y mysql-libs-5.1.73-7.el6.x86_64

    mysql-libs-5.1.73-7.el6.x86_64

    替换成

    rpm -qa | grep -i mysql 中显示的 mysql 名称

    上传 mysql

    这里可以使用 FileZilla 上传到自己的服务器上。

    安装依赖包(需要联网)

    yum -y install libaio.so.1 libgcc_s.so.1 libstdc++.so.6

    yum update libstdc++-4.4.7-4.el6.x86_64

    yum -y install libncurses.so.5 libtinfo.so.5

    解压 mysql

    注意此处没有 -xvf 没有 z，因为这个解压缩的包中没有.gz。

    tar -xvf MySQL-5.6.22-1.el6.i686.rpm-bundle.tar

    安装 server

    rpm -ivh MySQL-server-5.6.22-1.el6.i686.rpm

    如果安装不上，一般是依赖没有装上，安装依赖包即可。

    安装 client

    rpm -ivh MySQL-client-5.6.22-1.el6.i686.rpm

    查询 mysql 服务运行状态

    service mysql status

    启动 mysql 服务

    service mysql start

    查看 root 密码

    vim /root/.mysql_secret

    登录 root 账户

    mysql -uroot -p

    输入密码即可登录

    修改 root 密码

    set password = password(“123456”);

    系统启动时自动启动 mysql 服务

    加入到系统服务

    chkconfig –add mysql

    自动启动

    chkconfig mysql on

    查询列表

    chkconfig | grep -i mysql

    当 2，3，4，5 开启即可。

    开启远程访问

    设置远程访问

    grant all privileges on . to ‘root’ @’%’ identified by ‘123456’;

    flush privileges;

    防火墙打开 3306 端口

    /sbin/iptables -I INPUT -p tcp –dport 3306 -j ACCEPT

    将打开的端口的访问规则保存在文件中

    /etc/rc.d/init.d/iptables save

    查看机器开放的端口号

    /etc/init.d/iptables status

    看到有 3306 端口开放即可。
