【VMware 安装 Centos】

2018-07-30

在我前面一篇文章有讲过如何安装 VMware 之后，配置一些东西，使网络生效的一篇教程，这篇讲讲如何安装 Centos。

开启虚拟机

这篇教程是建立在看过我的上一篇博客的基础上写的。先开启此虚拟机。

选择安装项

选择第一项。Install or upgrade an existing system。

跳过检查

使用 tab 键，选择 skip，回车 enter。

选择语言环境

选择中文简体。

选择键盘类型

默认即可。

选择存储设备

默认选择基本存储设备即可

格式化硬盘

选择是，忽略所有数据。

设置网络

这里可以修改你的网络主机名，点击配置网络，点击编辑，勾选自动连接，以及在 IPv4 设置中使用自动（DHCP），点击应用。

设置时区

默认选择亚洲/上海即可。

设置密码

学习使用，123456 即可。

选择安装类型

使用所有空间。点击下一步，将修改写入磁盘。

选择系统安装类型

只有命令行的系统选择 Basic Server 即可。

安装

耐心等待即可。安装完成之后，选择重新引导即可。
