IDEA 比 Eclipse 好在哪？

2018-07-01

前段时间去面试【亚信公司】的时候，面试官问我为什么不用 Eclipse,而使用 IDEA，作为一个使用 IDEA 只有一年的码农，当时还真没给面试官说出来 IDEA 哪里好了。特此总结一下。

1.颜值高，谁说程序员就没有审美了？我就觉得 IDEA 嘿嘿的背景配上高亮的代码，挺符合我的审美的。

2.IDEA 中我再也不用手动保存代码了，再也不用担心某一天电脑突然断电或者蓝屏，代码没有保存了，简直是妈妈再也不用担心我的代码了。

3.IDEA 的代码补全比 Eclipse 快多了，比如说我按 psv + tab 键我就能快速生成一个 main 函数。

4.快捷键用起来简直不要太爽。比如说我在写一个 JavaBean 的时候，我需要写每个属性的 set 和 get 方法，在 Eclipse 中我要通过鼠标点点点，然后再选择需要 set 和 get 的属性，最后才生成。而在 IDEA 中，直接可以通过 alt + insert 生成代码。

5.对语言的支持太好了。不仅写 Java 时会有提示，而且写前端，它还是有提示，要是用 Eclipse,前端不仅没提示，而且还偶尔给我报个红叉，明明我代码没有一点问题的，嘤嘤嘤。

6.IDEA 的调试功能比 Eclipse 好太多了，好吧，语言有点匮乏了。在 IDEA 中只需要将光标移到你的表达式中然后按下 alt + f8。IDEA 会显示一个对话窗口，然后你可以对参数值进行修改，最后 IDEA 会帮你计算这个表达式的值。

当然 IDEA 还有更多的优势，留给大家慢慢挖掘。
