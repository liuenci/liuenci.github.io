总结:

  1. 写UserService类时需要写set方法，get方法可以不用，因为我们只需要注入对象，而且一般set注入属性会比较多，使用构造方法注入也可以，但一般使用set注入。
  2. 配置XML时注意要先注入UserDao的对象，然后注入需要调用UserDao的方法的对象，另外需要注意的就是属性中对象的值的标签不是用 value 了，而使用 ref (引用) 来注入对象属性。
