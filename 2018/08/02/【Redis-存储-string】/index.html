【Redis 存储 string】

2018-08-02

概述

字符串类型是 Redis 中最为基础的数据存储类型，他在 Redis 中是二进制安全的，这便意味着该类型存储和获取的数据相同。在 Redis 中字符串类型的 Value 最多可以容纳的数据长度是 512M。

常用命令

赋值

设置 key 持有指定的字符串 value，如果该 key 存在则进行覆盖操作，总是返回 ok。

set key value

取值

获取 key 的 value，如果该 key 不存在，返回 nil。

get key

删除

删除指定 key。

del key

数值增减

将指定的 key 的 value 原子性的递增 1，如果该 key 不存在，那么会先创建 key，默认初始值为 0，递增之后为 1。

incr key

将指定的 key 的 value 原子性的递减 1，如果该 key 不存在，那么会先创建 key，默认初始值为 0，递增之后为 -1。

decr key

扩展命令

增加固定的值

将指定的 key 的 value 原子性增加 num，如果该 key 不存在，那么会先创建 key，默认初始值为 0，在 incrby 之后，值为 num。

incrby key num

减少固定的值

将指定的 key 的 value 原子性减少 num，如果该 key 不存在，那么会先创建 key，默认初始值为 0，在 incrby 之后，值为 -num。

decrby key num

拼凑字符串

如果该 key 存在，则在原有的 value 后追加该值；如果该 key 不存在，则会创建一个 value 为 num 的 key。

append key num
