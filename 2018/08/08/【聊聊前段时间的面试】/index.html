【聊聊前段时间的面试】

2018-08-08

距离上一次推文已经有好几天的时间了，在没有推文的日子里，后台的数据里显示，每天都有新的读者关注我，这点我很是激动啊。关注的读者越多，自己得到的认同感就越强烈，就更有动力更新推文了。

熟悉我的读者都知道，我现在正处于积极找工作的状态，刚好现在也是秋招的预热期，很多大企业的提前批以及内推像漫天飞羽一样在各种招聘群里炸散开。我也顺势写好简历，拿着在各种招聘群里发布的内推码，在各大企业的官网进行网申。

果然，前几天，北京一家公司的 hr 在微信上联系了我，问我能不能参加他们公司的面试。那我当然愿意啊，正好多收割一个 offer，我对自己还是有信心的。

本来没打算把面试详情发在公众号里的，但是在朋友圈里说起这件事的时候，很多同学对面试时问到的问题比较感兴趣，所以考虑了一下，我还是发出来供大家参考吧。

面试官：简历上写着你在 xx(我实习的一个公司名)公司实习过，能聊聊你在公司里做的项目吗？

我：嗯嗯可以，然后聊项目 balabala。

面试官：简历上你说自己封装了一个高可复用的 servlet，谈谈你是怎么封装的吧，实现了哪些功能呢？

我：封装超类啊，编码转换啊，重定向带参啊，简化原生 servlet 方法啊。balabala。

面试官：有没有更好的办法能实现你上面所说的功能呢？

我：有的有的，过滤器也是可以实现这些功能的。

面试官：能聊聊过滤器的工作过程吗？

我：（心里 OS）md 不会，尴尬了， B 吹大了。（表面说）这个原理和 mvc 差不多，主要经过请求，然后过滤请求，执行具体方法，返回视图名，再经视图解析完成。

面试官：不对吧，你这跳的有点快。

我：嗯嗯，可能我记忆有偏差，加上这块研究不深。

面试官：没关系，谈谈编码转换吧，eclipse 里面生成的默认编码是什么编码？你自己转成了什么编码？

我：ISO 什么什么 -9 吧，具体不太记得了，然后转成了 utf-8。

面试官：我看你拿过一个省级的 Java 的比赛的奖是吧？那个比赛比的是些什么呢？

我：就是一个算法比赛，比方说给我一颗二叉树，如何实现它的镜像这种。

面试官：你在实习期间用过哪些 Java 的集合类呢？

我：List，Map 这类用的比较多。

面试官：嗯嗯，Map 是吧，好，我们来聊聊 HashMap 的源码吧？

我：（内心 OS：终于聊源码了，这个我很擅长）可以啊，没有问题。

面试官：HashMap 内部的结构是怎么样的？

我：分版本来看：Java7 和 Java8 两个版本的结构不一样，然后 balabala。

面试官：Java8 为什么要把 HashMap 的链表变成红黑树？

我：这个要看 hash 算法了，然后讲了哈希碰撞会怎么样？哈希算法性能差会怎么样？最后讲到如果不把链表改装成红黑树又会怎么样？balabala。

面试官：hashmap 里面有一个 hreadhold ，你知道是干嘛的吗？

我：然后我给他聊到了负载因子，外加初始位桶容量，超过 hreadhold 如何进行扩容，扩容又是怎么扩容的，为什么扩容是 * 2，又讲到位运算。

面试官：老版本里面的 HashMap 中的数组是链表指向数组，还是数组指向链表？

我：（内心 OS：md 不会，瞎猜一个）嗯嗯，这个是数组指向链表，然后 balabala 谈了一点我的想法。

面试官：那你懂二次 hash 过程吗？

我：听过这个，忘了，不过还是 balabala 的讲了一点自己的想法。

面试官：聊聊异常捕获过程吧。

我：balabala。

面试官：用过 maven 的哪些功能？

我：依赖导入啊，多模块开发啊，多种骨架开发啊，balabala。

面试官：听说你还会用 git 的命令行，聊聊你用过哪些命令行？

我：然后给他说了一套完整的仓库建立，项目上传，远程分支追踪，多分支如何开发，如何合并，如何解决冲突。balabala。

面试官：对并发了解多少？

我：说了一下对 JUC 下一些包的了解。

然后就被通知一面过了，总共面试时长 34 分钟，接下来就是二面了，二面聊的时间不长，只有 12 分钟，我都怀疑我挂掉了，主要面试过程问了我关于集合，多线程，网络，io，然后就是关于缓存这一块我对 redis 的使用程度，还有线程安全这一块，问了一些数据一致性的问题。

整体下来，两轮技术面让我觉得对方还是挺有技术含量的，设计的范围也比较广，聊的东西因为涉及到了源码的具体流程让我觉得问题还是比较的有深度。

最后 hr 在微信上直接就联系了我，和我说了一下公司的位置，以及薪酬福利，实习生入职就标配 mac，租房也有补贴，看起来在实习里面来说还是比较可以的。

不过我最后还是选择拒绝了，原因有三点。

  1. 现在秋招才刚刚开始，从现在的面试来看，我完全不怕我找不到工作，所以我可以挑选的机会比较的多
  2. 北京对我来说，感觉并没有太大的吸引力，相对于上海以及杭州来说，空气这一块还有待加强。
  3. 实习的时间过于长，听 hr 的意思是要实习到我毕业，一年的时间对我来说有点长，所以呢，我还是再找找吧。

秋招是一个比较锻炼人的机会，如果有时间，我个人认为可以去试试面试，这不仅仅是检验自己这几年学习的成果，也是一次和别人一起竞争的机会，更是一次认清自己技术能力的机会。

最后愿我的读者遍地 BAT。
