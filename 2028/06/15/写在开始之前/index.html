写在开始之前

By cier

2028-06-15 更新日期:2018-06-19

文章目录

首先算下我写博客以来迭代了几次版本。记得最开始的时候是在博客园上注册的博客账号，和面发现博客园的 seo 做的挺差的，在百度上进行搜索的时候，同样的文章基本上都搜索不到博客园上这个平台的文章。

后面换成了 CSDN 的博客平台，当时写博客的时候感觉用起来用户体验还是不错的，页面也还比较美观，当时还用过一段时间。

直到我后来开始专注于写作领域的时候，而不太在意页面排版的时候。我开始接触了 Markdown。那时候的 CSDN 好像还不支持 markdown 语法，再加上在别的平台上发文章会有很多的限制，就开始琢磨着自己搭载一个小窝。

也就后面有了我的博客的第三个迭代，阿里云配合 wordpress，另外自己在万网上购买了自己的第一个域名 liuenci.cn。然后自己连续半年都在自己的小窝上写文章，截止到 2018 年 6 月 10 号，我总共贡献了接近 80 篇文章，可惜现在全部清空了。

不过我还留有 markdown 文件，如果您对我的文章感兴趣，可以前往我的 github 查看，地址传送门:https://github.com/liuenci/BlogBackup 。里面有我遇到的前端，后端，数据库，产品等等的一些感想，或者问题的解决方案，有兴趣的同学可以给个 star。

再后来，我的云服务器到期了，是的，我开始被外物摁住了喉咙。哈哈哈，好吧，其实还有另外的原因，因为最近折腾了一个项目，项目需要上线，不想重新购买服务器的我只能拿我的阿里云 ECS 开刀了。

是的，现在的博客是您见到的第四个迭代版本，基于 github + hexo 的静态博客平台，搭建方便，支持 markdown,服务器由 github 提供，唯一不足的是 访问可能稍微的有点慢，不过不影响了，最终的结果是我还有一个小窝可以写写东西。

这个博客目前刚刚开始，接下来的时间我会将以前自己写过的一些好的文章迁移过来，争取做一个高质量的博客，另外微信公众号上一些我觉得不错的文章我也会同步到博客上。
